EXP 01

Password Based Door Lock System

Arduino Uno · 4×4 Keypad · Servo Motor · Bolt Lock

Aim
To design and implement a Password Based Door Lock System using Arduino Uno and a 4×4 keypad, where a servo motor physically operates a bolt/barrel lock mechanism. The system unlocks the bolt when the correct password is entered and automatically re-locks after a few seconds.
Theory
A Password Based Door Lock System controls physical access using a predefined password.

Arduino Uno: Microcontroller that reads keypad input and drives the servo motor.

4×4 Keypad: Matrix keypad with 16 keys for password entry.

Servo Motor (SG90): Rotates a lever arm connected to the bolt — 0° = locked, 90° = unlocked.

Barrel/Bolt Lock: A physical steel bolt lock operated by the servo arm via a linkage wire.

LCD Display (16×2): Shows system status — "Enter Password", "Access Granted", "Wrong Password".

When correct password + # is pressed, servo rotates to 90° pulling the bolt open. After 4 seconds the servo returns to 0° locking the bolt again.

Procedure
  1. 1Mount Arduino Uno, servo motor and bolt lock on a cardboard base as shown in hardware photo.
  2. 2Connect the 4×4 keypad row pins to Arduino digital pins 9, 8, 7, 6 and column pins to 5, 4, 3, 2.
  3. 3Connect the 16×2 LCD — RS to pin 13, EN to pin 12, D4–D7 to pins 11, 10, A0, A1.
  4. 4Connect potentiometer wiper to LCD V0 pin for contrast control.
  5. 5Connect servo signal wire to Arduino pin A2, red to 5V, black to GND.
  6. 6Attach a linkage wire from the servo arm to the bolt lock slider.
  7. 7Open Arduino IDE, install Keypad and Servo libraries, write and upload the code.
  8. 8Enter correct password + # — servo rotates 90°, bolt retracts, access granted.
  9. 9After 4 seconds servo returns to 0°, bolt extends and locks again automatically.
  10. 10Wrong password — servo stays at 0°, bolt remains locked, LCD shows error.

Complete Hardware Setup
Arduino Uno connected to a 4×4 membrane keypad and SG90 servo motor. The servo arm is linked via wire to a physical barrel bolt lock mounted on a cardboard base — exactly as simulated above.
Arduino Uno SG90 Servo Bolt Lock 4×4 Keypad
🔩 Bolt Lock Mechanism
The silver barrel bolt lock is the actual locking component. When the servo motor rotates 90°, the arm pulls the bolt slider open via a linkage wire, allowing the door to open. At 0°, the bolt extends and physically prevents opening.
⚙️ Servo Motor (SG90)
The Tower Pro SG90 micro servo provides the rotational force to operate the bolt. Signal wire connects to Arduino pin A2. Red = 5V, Brown/Black = GND. Servo rotates between 0° (locked) and 90° (unlocked).
⌨️ Keypad Wiring
The 4×4 membrane keypad uses 8 wires — 4 rows + 4 columns connected to Arduino digital pins 2–9. The flat ribbon cable connects directly to the Arduino via female jumper wires.
Interactive Wiring Lab
How to use: Click any pin dot to select (turns white), then click another pin to draw a wire. ✅ Correct = colored wire  |  ❌ Wrong = red dashed wire. Press Check Connections to validate all.
